## Gerenciamento de Conteúdo

Joomla! é um sistema de gerenciamento de conteúdo em essência e possui excelentes recursos para facilitar a organização e o gerenciamento de conteúdo. O editor WYSIWYG embutido permite que os usuários editem conteúdo sem qualquer conhecimento de código. Existem muitas maneiras possíveis de exibir o conteúdo do site com diferentes layouts e módulos pré-instalados para mostrar os artigos mais populares, itens mais recentes, artigos relacionados e mais.

## Artigos

Um artigo é uma unidade de informação exibida de forma proeminente no centro de uma página web. Adicionar uma imagem é tão fácil quanto arrastar e soltar do seu computador diretamente no editor de conteúdo. Isso funciona em qualquer lugar onde você está usando o editor padrão TinyMCE WYSIWYG. E se você quiser adicionar um módulo em um artigo, o Joomla! torna isso realmente fácil com um botão na barra de ferramentas do editor. Você não precisa aprender nenhuma sintaxe ou lembrar os detalhes do módulo, pois a interface do usuário faz isso por você.

## Categorias

Quando você está gerenciando conteúdo, a organização é um requisito fundamental. Ser capaz de criar categorias com subníveis e sem limites de profundidade é uma grande vantagem para ajudar a gerenciar grandes sites.

## Tags

Quando a categorização não é suficiente para estruturar o conteúdo, é hora de considerar uma estrutura organizacional plana, que é melhor atendida por meio de tags. Além disso, a marcação no Joomla! também suporta aninhamento, então os limites simplesmente não existem! Comece a etiquetar.

## Pesquisa Inteligente

Com a Pesquisa Inteligente integrada, os visitantes do site poderão encontrar rápida e facilmente as informações apropriadas do site. E ainda mais, graças às estatísticas, você pode analisar as necessidades dos seus visitantes e ajustar seu conteúdo ainda melhor para atendê-los. Você tem a capacidade de usar a indexação inteligente incluída, opções de busca avançadas, sugestões automáticas de busca - tornando a pesquisa do Joomla! a melhor da categoria desde o início.

## Multilíngue

Joomla! é a plataforma CMS de código aberto multilíngue mais popular e amplamente suportada no mundo, oferecendo mais de 64 idiomas. Webmasters e criadores de conteúdo podem criar sites para serem apresentados em vários idiomas, sem nunca precisar sair das opções disponíveis no software principal do Joomla!. Este é um grande avanço e representa um conjunto de capacidades que pode tornar os sites muito mais acessíveis, alcançando um público muito maior.

## Responsivo com Bootstrap

Joomla! está pronto para dispositivos móveis e permite que você construa mais do que apenas sites, mas aplicações online que podem responder a praticamente qualquer dispositivo. Os templates principais do Joomla! são construídos com Bootstrap, o que os torna responsivos desde o início. Isso significa que você tem um conjunto de ferramentas para trabalhar que torna a criação de templates ainda mais fácil!

## Fontes Bonitas para Aquele Toque Extra

Designers conhecem o poder das fontes para expressar ideias e estratégias de design. Com o Joomla! você não precisa se limitar às fontes padrão da web. O núcleo do Joomla! por si só abre um novo mundo de expressão, pois permite a liberdade de usar fontes do Google para tornar realidade aquele novo sonho de design!

## Edição no Frontend

Editar conteúdo deve ser fácil e rápido. Você está lendo o conteúdo do seu site e vê uma mudança que precisa fazer. Não é mais necessário fazer login no Backend para edições simples de conteúdo e módulos. Simplesmente faça login no Frontend, clique e edite lá.

## Realce de Sintaxe

Às vezes, a visualização do editor WYSIWYG não é suficiente e você precisa alternar para a visualização do código HTML. No Joomla! todo o código é destacado na visualização do código-fonte dentro do editor TinyMCE embutido. Isso tornará muito mais fácil ler todas as suas tags HTML e classes CSS. A marcação é exibida de forma clara. Além disso, você pode buscar e substituir diretamente na visualização do código HTML.

## Controle de Versão de Conteúdo

Você nunca mais perderá uma versão importante anterior do seu artigo e outras alterações no seu site. Agora você pode rastrear exatamente quem fez quais alterações, quando e, se uma nota de versão foi inserida, por que o item foi editado. Então, se necessário, você pode reverter para qualquer versão anterior.

## Fluxos de Trabalho

O componente de Fluxo de Trabalho de Publicação é usado por uma equipe de usuários com diferentes responsabilidades para mover um artigo de uma etapa do processo de publicação para a próxima. Por exemplo: um grupo de usuários pode compor posts de blog, mas não publicá-los; um segundo grupo de usuários aprova e publica os posts de blog. O fluxo de trabalho pode envolver mais etapas. Se apenas um usuário faz tudo, então um fluxo de trabalho não é necessário.

## Verificador de Acessibilidade

Todo visitante que não conseguir visualizar sua página é potencialmente um cliente perdido. O Joomla! permite que os autores criem conteúdo acessível de qualidade com um plugin para destacar visualmente problemas comuns de acessibilidade e usabilidade.

## Atalhos de Teclado

Existem 9 atalhos de teclado incorporados no lado do administrador do seu site, como J + F para pular para o campo de busca ou J + S para salvar. Pressione J + X para obter uma lista de todos. As combinações de teclas são todas sequenciais, tornando-as mais acessíveis do que pressioná-las simultaneamente.

## Gerenciamento de Mídia

O componente de Mídia é utilizado para fazer upload, organizar e gerenciar arquivos e pastas de mídia. Ele possui configurações MIME configuráveis para lidar com uma ampla variedade de tipos de mídia. O componente de Mídia está integrado ao Editor de Artigos para permitir o acesso a imagens e todos os outros arquivos de mídia, facilitando o uso e aprimoramento do conteúdo escrito.

## Gerenciamento de Menu

O sistema de Menu permite a criação de quantos menus e itens de menu forem necessários. Você pode estruturar sua hierarquia de menus (e itens de menu aninhados) completamente independente da estrutura de conteúdo. Coloque um menu em vários lugares e no estilo que desejar; utilize rollovers, dropdown, fly outs e quase qualquer outro sistema de navegação que você imaginar. Além disso, migalhas de pão automáticas são geradas para ajudar os usuários a navegar no seu site.

## Gerenciamento de Usuários

Pronto para uso, o Joomla! possui nove grupos de usuários com vários tipos de permissões sobre o que os usuários atribuídos a um grupo específico podem acessar, editar, publicar e administrar usando as Listas de Controle de Acesso (ACL). O ACL do Joomla! é extremamente poderoso e pode ser configurado para atender a qualquer necessidade que você possa ter.

Joomla! possui um sistema de registro que permite aos usuários configurar opções pessoais, como senha, nome, endereço de email, editor padrão e assim por diante.

A autenticação é uma parte importante do gerenciamento de usuários e o Joomla! suporta múltiplos protocolos, incluindo LDAP, OpenID e até mesmo Gmail. Isso permite que os usuários usem as informações de suas contas existentes para simplificar o processo de registro. Tudo isso pode ser expandido com extensões, proporcionando controle total sobre o que os usuários podem acessar e como eles se autenticam no seu site.

Os administradores podem se comunicar rápida e eficientemente com os usuários individualmente por meio de mensagens privadas ou com todos os usuários do site através do sistema de envio em massa de e-mails.

## Autenticação Multifator

Joomla! foi o primeiro CMS importante a implementar a autenticação de dois fatores. Agora possui autenticação multifatorial, permitindo que os usuários escolham um ou mais métodos de autenticação além do método tradicional de nome de usuário e senha. Um código de uso único recebido no seu celular ou um Yubikey são apenas dois dos métodos disponíveis para tornar seu site ainda mais à prova de invasões.

## WebAuthn

WebAuthn está disponível para login sem senha usando um dispositivo de hardware ou software.

## Gerenciamento de Templates

Modelos no Joomla! são mais do que uma estrutura para gerenciar seus designs, mas sim um conjunto de ferramentas poderoso para fazer seu site parecer exatamente como você deseja. Você tem controle total sobre sua apresentação, já que pode usar um único modelo para o site inteiro ou um modelo separado para cada seção do site ou item de menu.

Com um sistema de substituição altamente avançado, os designers obtêm uma quantidade incrível de controle sobre como as páginas e elementos das páginas são apresentados sem tocar em nenhum código principal! Praticamente qualquer HTML gerado pelo Joomla! pode ser personalizado para o seu projeto.

Os Modelos Filhos permitem que você faça alterações em um modelo que não serão substituídas se houver uma atualização do modelo. As alterações são feitas em um clone parcial do modelo original.

## Gestão de Banner

Com o gerenciador de banners, você tem a possibilidade de adicionar facilmente publicidade e monetizar seu site. A ferramenta permite criar clientes e campanhas, adicionar quantos banners você precisar, até mesmo códigos personalizados, definir números de impressões, rastrear os cliques e muito mais...

## Gerenciamento de Contatos

Não é suficiente ter apenas um formulário de contato em seu site? O componente de contatos permite que você adicione vários contatos, departamentos e categorias, e estenda as informações básicas de contato com informações diversas e uma imagem. Configure facilmente um formulário de contato para cada contato que você criar e permita o acesso ao público ou apenas para alguns usuários registrados, ou crie uma lista desses contatos.

## Sindicação e Gerenciamento de Feed de Notícias

Certifique-se de que seus visitantes fiquem atualizados sobre o novo conteúdo que você está adicionando, mesmo quando eles vêm apenas de vez em quando. Com a Sindicação você cria um feed ao qual os usuários se inscrevem no leitor de RSS favorito e assim recebem as atualizações. Com o gerenciamento de feed de notícias, você pode integrar feeds RSS ao seu site. Reúna todas as postagens de alguns dos maiores sites de notícias e mostre-as em seu site, por exemplo.

## Ajuda do Administrador

A maioria das páginas do administrador tem um botão de Ajuda no canto superior direito. Ele abre uma janela pop-up contendo informações sobre todas as opções naquela página. Algumas páginas também têm um botão de Ajuda Inline que alterna entre ligar/desligar o texto explicativo abaixo dos campos selecionados nos formulários de entrada de dados do administrador. Há uma página de links de Ajuda, incluindo um para todas as páginas de Ajuda e muitos para recursos online, como a Documentação Joomla! e Fóruns de Usuários.

## Atualizações Fáceis

Joomla! possui um recurso de *Atualização de Versão com Um Clique* para facilitar a atualização para usuários de qualquer nível de habilidade. O atualizador embutido tem um verificador automático que notifica você se algo precisa ser atualizado; isso inclui notificações do software principal e das extensões Joomla! que usam este recurso. O plugin de notificação de atualização por email verifica a disponibilidade de novas versões do Joomla! e envia aos Super Usuários do site uma notificação por email.

## Extensibilidade Poderosa

O núcleo do Joomla! é apenas o começo, o verdadeiro poder está na forma como você pode personalizar o Joomla. Mais de 8000 Extensões estão disponíveis para expandir seu site e ampliar sua funcionalidade. Visite o [Joomla Extensions Directory (JED)](https://extensions.joomla.org) para ver milhares de maneiras de aprimorar o Joomla! de acordo com suas necessidades.

## Agendador de Tarefas

Tarefas que você realiza repetidamente ou tarefas futuras que não devem ser esquecidas podem ser automatizadas com o Agendador de Tarefas.

## Funcionalidades do Sistema

Carregamentos de página rápidos são possíveis com cache de página, cache de módulo em nível granular e compressão de página GZIP.

Se o administrador de sistema precisar solucionar um problema, as opções de depuração e relatório de erros integradas são inestimáveis.
